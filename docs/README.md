# Igniter Ruby Documentation

Status: split-era baseline index

This repo is the Ruby Framework lane for Igniter. It carries the framework
package surface and selected current documentation, not the Igniter-Lang
specification and not lab/frontier experiments.

## Boundary

Igniter Ruby does not own Igniter-Lang grammar/spec/runtime authority. Language
ideas should enter through explicit bridge notes or package-local integration
work, not by silently importing language claims into framework docs.

Guide/dev docs and runnable examples are deferred until curated transfer
decides which ones still carry value.
